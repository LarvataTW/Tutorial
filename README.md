# 關於

這份文件要說是教育訓練手冊、就職手冊、員工訓練、自我學習教程，都可以。   
目的是通過完成此文件安排的學習內容，訓練讀者成為一位具有基本職業素養的資訊產業工作者。

文件內容與其說是教程，不如說是各種學習主題的教程集合與安排，
因為公司畢竟不是學校，或專門做培訓的教學機構，
網路上有更專業、質量優良的教學材料、影片，
收集過濾優質的教程來使用，而不用重造輪子，
應該是更有效率的作法。
